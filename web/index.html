Qanday ishlatiladi

  1. DMG'ni oching va ilovani Applications'ga torting
  2. Ishga tushiring → mikrofon va Accessibility ruxsatlarini bering
  3. Istalgan joyda ⌃⌥D → 🔴 gapiring → ⌃⌥D
  4. Matn kursor turgan joyga yoziladi ✍️
⚠️ Birinchi ochishda ogohlantirish chiqsa

Ilova Developer ID bilan imzolangan, lekin hali Apple notarizatsiyasidan o'tmagan. Birinchi marta:

  • macOS 13–14: ilovaga o'ng tugma (Control-click) → Open → Open
  • macOS 15: System Settings → Privacy & Security → pastga → «Open Anyway / Все равно открыть»

Bir marta shunday qilsangiz, keyin doim normal ochiladi.
